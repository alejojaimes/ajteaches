# ajteaches

Personal technical blog by Alejandro Jaimes. Engineering, taught well.

## Stack

- Next.js 16 (App Router) + TypeScript
- Tailwind CSS v4 + Shadcn UI
- PostgreSQL (Neon) + Prisma
- Clerk (auth) + Cloudinary (images) + Resend (emails)
- Tiptap (editor)
- Deployed on Vercel at ajteaches.dev

## Local development

\`\`\`bash
npm install
cp .env.example .env.local

# Fill in .env.local with your keys

npm run dev
\`\`\`

Open http://localhost:3000 to see the result.
